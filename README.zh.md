# SpaceSwitcher

一款 macOS 菜单栏应用，可通过鼠标手势切换 Spaces 工作区及触发 Mission Control。

[English](README.md)

---

## 下载

克隆或下载本仓库，直接双击 **SpaceSwitcher.app** 运行。

> **首次打开时 macOS 可能提示"来自身份不明的开发者"。**
> 右键（或按住 Control 单击）→ **打开**，在弹窗中再次点击 **打开** 即可。

---

## 系统要求

- macOS 12 Monterey 或更高版本
- **辅助功能**权限（系统设置 → 隐私与安全性 → 辅助功能）
- **输入监控**权限（鼠标按键测试窗口所需）

---

## 首次启动

1. 双击 **SpaceSwitcher.app**（如被 Gatekeeper 阻止，右键 → 打开）。
2. macOS 会请求**辅助功能**权限，在系统设置中授权。
3. 重新启动应用，菜单栏中会出现光标箭头图标（↖）。

---

## 菜单栏

点击 ↖ 图标打开菜单。

| 菜单项 | 操作 |
|--------|------|
| Space Left | 切换到左侧工作区（Ctrl+←） |
| Space Right | 切换到右侧工作区（Ctrl+→） |
| Gesture Remap… | 打开手势设置 |
| Test Mouse Buttons… | 打开按键调试窗口 |
| Quit | 退出应用 |

---

## 手势重映射

从菜单栏打开 **Gesture Remap…**。

### 基本设置

1. **Enable gesture remap** — 勾选以启用手势重映射。
2. **Mouse** — 选择特定鼠标，或保持 *Any Mouse* 以匹配所有设备。
3. **Button** — 按住触发手势的按键（默认：Button 4，通常为"后退"键）。

### 灵敏度调节

| 滑块 | 说明 |
|------|------|
| Distance（距离） | 触发切换所需的移动距离（像素）。数值越大越不灵敏。 |
| Speed（速度） | 触发快速滑动所需的速度（像素/秒），无需达到距离阈值即可触发。数值越大越不灵敏。 |

### 手势说明

按住配置好的按键，同时移动鼠标：

| 方向 | 动作 |
|------|------|
| 向左滑 | 切换到左侧工作区（Ctrl+←） |
| 向右滑 | 切换到右侧工作区（Ctrl+→） |
| 向上滑 | Mission Control / 上方工作区（Ctrl+↑） |
| 向下滑 | App Exposé / 下方工作区（Ctrl+↓） |

**短按**（按住不足 0.3 秒且未滑动）：原始按键点击将正常透传。

设置会自动保存，下次启动时自动恢复。

---

## 鼠标按键测试窗口

打开 **Test Mouse Buttons…** 可查看鼠标的原始 HID 事件，便于确认物理按键对应的按键编号。

---

## 权限说明

| 权限 | 用途 |
|------|------|
| 辅助功能 | 允许应用发送按键事件（Ctrl+方向键）切换工作区，并通过 CGEventTap 拦截/屏蔽鼠标按键。 |
| 输入监控 | 允许应用读取鼠标的原始 HID 事件，用于测试窗口和设备识别。 |
